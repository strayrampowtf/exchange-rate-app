# Курсы валют
Проект "Курсы валют" реализован в рамках университетского курса "Технологии создания и поддержки ПО". 
## Краткая характеристика проекта
Проект состоит из:
  - PWA-приложения, представленного исходным кодом;
  
  - User story и use case в виде текстового описания;
  
  - Use case diagram;
  
  - Прототипа главной страницы приложения в figma;
  
  - Техническое задание проекта.

Создано PWA-приложение для отслеживания курсов валют ЦБ РФ с поддержкой офлайн-режима, конвертером и графиками.

Дизайн вдохновлен приложением «Акции» от Apple.

## Основные возможности

   - Отображение актуальных курсов валют от официального API ЦБ РФ
   
   - Поиск по всем валютам
   
   - Встроенный конвертер валют (включая RUB)
   
   - Графики изменения курса за неделю, месяц, год
   
   - Реальное изменение за 24 часа (на основе кэширования предыдущих значений)
   
   - Полностью тёмная тема в стиле iOS (SF Pro, чёрный фон #000000)
   
   - Полная адаптивность от 320px
   
   - Модальные окна вместо переходов между страницами (как в нативных iOS-приложениях)
   
   - PWA — устанавливается на устройство, работает офлайн
   
   - Офлайн-режим — кэширование API-запросов через Service Worker.
   
##  Технологии

  - Frontend: React + TypeScript + Vite

  - UI: CSS Modules, Recharts (графики)

  - Backend: Node.js + Express (прокси к API ЦБ РФ)

  - Деплой: Фронтенд → GitHub Pages (с CI/CD через GitHub Actions)

  - Бэкенд → Render (в инактиве)
  - API: ЦБ РФ XML_daily.asp
